## Description

Console application made with C++ with 5 of the most common data structures:

- Stack
- Queue
- Hash Table
- Binary Search Tree
- Graph

## Installation

- Windows

Install C++ via VisualStudio Installer or MinGW

- Linux

Just open terminal, go to folder DataStrucutres and run:

- MacOS

Install C++ via XCode IDE

```bash
$ g++ *.cpp
```
